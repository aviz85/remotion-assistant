# Kinetic Video Creator Setup

Build me an automated kinetic typography video creation system using Claude Code skills.

## Overview
I want to create professional kinetic typography videos where:
1. I provide an idea/topic
2. Claude writes an emotionally-crafted script with bracket directions
3. Speech is generated with ElevenLabs TTS
4. Music is generated to match the duration and emotional arc
5. Audio is transcribed for word-level timing
6. Remotion creates animated text synced to speech
7. Final video is rendered and optionally uploaded to YouTube

## Required Dependencies

### 1. Install Remotion
```bash
npm create video@latest
# or in existing project:
npm install remotion @remotion/cli @remotion/player
```

### 2. Install Remotion Skills
```bash
npx @anthropic-ai/claude-code skill add remotion-dev/skills/remotion-best-practices
```

**IMPORTANT:** After installing skills, restart Claude Code (exit and re-enter the project) to load the new skills!

### 3. ElevenLabs API Setup
Create `.env` file:
```
ELEVENLABS_API_KEY=your_api_key_here
```

Required for:
- Text-to-Speech generation
- Music generation (specify exact duration)
- Scribe v2 transcription (word-level timing)

### 4. FFmpeg (for audio mixing)
```bash
# macOS
brew install ffmpeg

# Ubuntu/Debian
sudo apt install ffmpeg
```

## Script Writing Guidelines

Write scripts with emotional bracket directions:
```
[dramatic pause] The future isn't coming.
[slowly, with weight] It's already here.
[building intensity] Every day, every decision...
[powerful, emphatic] ...shapes the world we'll live in.
[warm, inspiring] And you? You're part of it.
```

Supported directions:
- Pacing: [pause], [long pause], [slowly], [faster]
- Tone: [whisper], [emphatic], [warm], [dramatic]
- Dynamic: [building], [descending], [with weight]

Word count guide:
- 30 sec = 60-80 words
- 60 sec = 120-150 words
- 90 sec = 180-220 words

## Pipeline Commands

### Generate Speech
```bash
cd ~/.claude/skills/speech-generator/scripts
npx ts-node generate_speech.ts -f script.txt -o speech.mp3
```

### Transcribe with Timing
```bash
cd ~/.claude/skills/transcribe/scripts
npx ts-node transcribe.ts -i speech.mp3 -o transcript.json --json
```

### Generate Music
```bash
cd ~/.claude/skills/music-generator/scripts
npx ts-node generate_music.ts \
  --prompt "Inspirational, cinematic, building" \
  --duration 90 \
  --output music.mp3
```

### Mix Audio (FFmpeg)
```bash
ffmpeg -y \
  -i speech.mp3 \
  -i music.mp3 \
  -filter_complex "[0:a]volume=1.0[speech];[1:a]volume=0.17[music];[speech][music]amix=inputs=2:duration=first[out]" \
  -map "[out]" -c:a libmp3lame -q:a 2 \
  final_audio.mp3
```

### Render Video
```bash
npx remotion render src/Root.tsx MyComposition output.mp4
```

### Upload to YouTube (optional)
```bash
cd ~/.claude/skills/youtube-uploader/scripts
npx ts-node youtube-upload.ts \
  --video output.mp4 \
  --title "Your Title" \
  --description "Description" \
  --tags "tag1,tag2" \
  --privacy unlisted
```

## Animation Modes

### Word Cloud (default)
Multiple words appear together, grouped by timing gaps.
Best for: Fast-paced, dynamic content.

### Single Word
One word at a time, centered.
Best for: Dramatic, impactful statements.

## Visual Customization

Available props:
- heroFontSize: Size for emphasis words
- strongFontSize: Size for important words
- normalFontSize: Size for regular words
- glowIntensity: 0-3 (glow around text)
- dustEnabled: Particle effects
- lightBeamsEnabled: Light beam animations
- colorScheme: 0-7 (predefined color palettes)
